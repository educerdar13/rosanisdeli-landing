## Design System Generated

- **Colors:** Near-black (#0F0F0F), Off-white (#F2EDE8), Muted Gold (#C4A35A), Charcoal (#2A2A2A)
- **Headline font:** Instrument Serif (refined, editorial)
- **Body font:** Manrope (geometric, clean contrast)
- **Layout:** Editorial-style with large typography. Hero is mostly text with subtle gold accent line. Work section uses a large-format grid. Minimal navigation.

## Sections Built

1. **Nav** — Name (left), Work / About / Contact (right), minimal
2. **Hero** — "Your brand, distilled." in large Instrument Serif + "Brand identity design for startups that want to mean something." + "See my work" text link
3. **Selected work** — 4 project cards in a 2x2 grid with hover reveal (project name + client)
4. **Services** — Horizontal scroll or stacked list: Brand Identity, Logo Systems, Visual Language, Packaging
5. **Testimonial** — Marco's quote on gold-accented background
6. **Contact** — "Have a project? Let's talk." + email link + social links
7. **Footer** — Minimal, credit line
