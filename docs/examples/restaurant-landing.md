## Design System Generated

- **Colors:** Terracotta (#C4703F), Cream (#FAF5EF), Warm Brown (#5C3D2E), Sage (#8B9E7E)
- **Headline font:** Cormorant Garamond (serif, elegant but warm)
- **Body font:** DM Sans (clean, readable)
- **Layout:** Asymmetric hero with large headline left, decorative element right. Menu highlights in a masonry-style grid. Testimonial as a single large quote.

## Sections Built

1. **Nav** — Logo (text), Menu, Location, Contact
2. **Hero** — "Slow bread. Good people." + subline about sourdough + "See Our Menu" button
3. **About strip** — One sentence about the bakery with a warm background
4. **Menu highlights** — 4 items in a staggered grid, not identical cards
5. **Testimonial** — Single large quote with name
6. **Visit section** — Address, hours, map placeholder, "Get Directions" button
7. **Footer** — Social links, credit line
